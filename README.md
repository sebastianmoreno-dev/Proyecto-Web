# 🏡 Proyecto Web de Bienes Raíces

Aplicación web full-stack para la gestión y visualización de propiedades inmobiliarias.

🔗 **Prototipo (Google Stitch):**
https://stitch.withgoogle.com/projects/5106793294907978005

---

## 📌 Descripción

Este proyecto fue desarrollado como parte de la materia **Tecnologías para el Desarrollo de Aplicaciones Web**.

La aplicación permite a los usuarios:

- 🏠 Visualizar propiedades disponibles
- 📋 Consultar detalles de cada inmueble
- 💰 Ver precios y características
- 📍 Conocer la ubicación de las propiedades

---

## 🏗️ Arquitectura del proyecto

El sistema está dividido en tres partes principales:

- 🎨 **Frontend:** Interfaz de usuario
- ⚙️ **Backend:** API REST
- 🗄️ **Base de datos:** Almacenamiento de información

---

## ⚙️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MySQL

---

## 🚀 Instalación y ejecución

### 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/tuusuario/proyecto-web.git
cd proyecto-web
```

---

### 2️⃣ Configurar backend

```bash
cd backend
npm install
node server.js
```

El servidor se ejecutará en:
👉 http://localhost:3000

---

### 3️⃣ Configurar base de datos

1. Crear base de datos en MySQL:

```sql
CREATE DATABASE inmobiliaria;
```

2. Ejecutar el archivo:

```bash
database/schema.sql
```

---

### 4️⃣ Ejecutar frontend

Abrir el archivo:

```bash
frontend/index.html
```

en tu navegador.
